# Smart AI & IoT Based Security System

## Project Description
This project aims to design an intelligent security system that integrates
Artificial Intelligence (AI) and Internet of Things (IoT) technologies to
protect modern data center environments.

## Team Structure
- AI Subsystem: Network intrusion detection, computer vision
- IoT Subsystem: Environmental monitoring, access control
- Network Subsystem: Network topology, firewall, routing
